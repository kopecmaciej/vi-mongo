# Release v0.1.33

<img src="https://github.com/kopecmaciej/vi-mongo/blob/master/assets/logo/no-background.svg" align="center"/>

## What's New

### Features

- **Configurable Database Panel Size**: Added ability to configure the "expanded" databases panel size through a dedicated UI configuration option [#105](https://github.com/kopecmaciej/vi-mongo/pull/105)
- **BSON Type Autocomplete Keywords**: Added autocomplete support for BSON types including `ISODate()`, `NumberDecimal()`, `NumberLong()`, `NumberInt()`, `Timestamp()`, `BinData()`, `MinKey`, `MaxKey`, and `RegExp()` [#108](https://github.com/kopecmaciej/vi-mongo/pull/108)

### Bug Fixes

- **DateTime Parsing**: Added support for multiple time formats when parsing DateTime values, fixing issues with non-standard date formats [#107](https://github.com/kopecmaciej/vi-mongo/pull/107)
- **Single Quote Queries**: Queries using single quotes are now properly handled by replacing them with double quotes before processing [#105](https://github.com/kopecmaciej/vi-mongo/pull/105)
- **Number Keywords**: Fixed handling of number keywords and improved date format display [#108](https://github.com/kopecmaciej/vi-mongo/pull/108)

### Improvements

- **Testing**: Added and updated tests for parser and regexp utilities [#108](https://github.com/kopecmaciej/vi-mongo/pull/108)
- **Code Quality**: Refactored regexp utilities and improved keyword handling in MongoDB parser

## Thank You

Thank you to everyone who helped improve this project!
