# Release v0.2.3

<img src="https://github.com/kopecmaciej/vi-mongo/blob/master/assets/logo/no-background.svg" align="center"/>

## What's New

### Bug Fixes

- **Config File Permissions** (by [@vitor-mariano](https://github.com/vitor-mariano), [#121](https://github.com/kopecmaciej/vi-mongo/pull/121)): Config files created before the `0600` mode was introduced kept their original (world-readable) permissions because `os.WriteFile` only applies the mode on creation. Vi Mongo now `chmod`s existing config files to `0600` on load.
- **Enable Paste & Default Exit Key**: The `EnablePaste` option was missing from the TUI initialisation, which could interfere with pasting in some terminals. The implicit `q` exit binding has also been removed to avoid exiting by accident.
