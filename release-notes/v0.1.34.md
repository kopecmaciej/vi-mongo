# Release v0.1.34

<img src="https://github.com/kopecmaciej/vi-mongo/blob/master/assets/logo/no-background.svg" align="center"/>

## What's New

### Features

- **Help Page Scrollbar**: Added a scrollbar to the help page for easier navigation through keybindings [#110](https://github.com/kopecmaciej/vi-mongo/pull/110)
- **Clear Filter Key**: Added a dedicated key binding to clear the database filter bar [#109](https://github.com/kopecmaciej/vi-mongo/pull/109)

### Bug Fixes

- **Database Filter Re-fetching**: Fixed an issue where filtering databases would unnecessarily re-fetch data [#109](https://github.com/kopecmaciej/vi-mongo/pull/109)
- **Filter Bar Text Clear**: Fixed missing text clear when resetting the filter bar [#109](https://github.com/kopecmaciej/vi-mongo/pull/109)

### Improvements

- **New Styles**: Updated and extended styles across all built-in themes (default, dark-blue, dracula, light-forest, light) [#110](https://github.com/kopecmaciej/vi-mongo/pull/110)

## Thank You

Thank you to everyone who helped improve this project!
